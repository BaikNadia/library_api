# Library Management API

REST API для управления библиотекой с использованием Django Rest Framework.

## Функциональность

- ✅ JWT аутентификация
- ✅ Управление пользователями, авторами, книгами
- ✅ Система выдачи книг
- ✅ Ролевая модель (читатель, библиотекарь, администратор)
- ✅ Документация API (Swagger/ReDoc)
